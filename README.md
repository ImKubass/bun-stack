# bun-stack

## Editor setup

Shared formatting is configured for both VS Code and JetBrains IDEs.

### VS Code

Use the recommended extensions from the workspace. The project settings in .vscode set Biome for JavaScript and TypeScript and Prettier for SCSS.

### PhpStorm / IntelliJ IDEA

JetBrains IDEs will automatically read .editorconfig and .prettierrc from the repository.

Recommended setup:

- use Prettier for SCSS formatting
- optionally install the Biome plugin for JavaScript and TypeScript
- enable format on save in the IDE if you want the same workflow as VS Code
